📘 Sécurité Facebook

Comment sécuriser son compte Facebook

Par Informa-Technique R  ·  11 mai 2026  ·  8 min de lecture

Facebook est l'un des réseaux sociaux les plus ciblés par les hackers. Un compte piraté peut être utilisé pour escroquer vos amis, diffuser de fausses informations ou accéder à d'autres comptes liés. En France, des milliers de comptes sont compromis chaque jour. Avec ce guide, vous allez sécuriser votre compte Facebook en moins de 15 minutes.

1. Activez la double authentification (2FA)

C'est la mesure la plus importante. Même si quelqu'un vole votre mot de passe, il ne pourra pas se connecter sans le second code de vérification.

  1. Allez dans Paramètres → Sécurité et connexion
  2. Faites défiler jusqu'à "Authentification à deux facteurs"
  3. Cliquez sur "Modifier"
  4. Choisissez "Application d'authentification" (Google Authenticator ou Authy)
  5. Scannez le QR code avec l'application
  6. Entrez le code à 6 chiffres pour confirmer
💡 Conseil :

Préférez une application d'authentification plutôt que le SMS. Les SMS peuvent être interceptés par une attaque SIM swapping, contrairement aux codes d'application.

2. Utilisez un mot de passe fort et unique

Votre mot de passe Facebook doit être différent de tous vos autres mots de passe. Si un autre site se fait pirater et que vous utilisez le même mot de passe, votre Facebook sera automatiquement compromis.

Pour changer votre mot de passe :

  1. Allez dans Paramètres → Sécurité et connexion
  2. Cliquez sur "Changer le mot de passe"
  3. Créez un mot de passe d'au moins 12 caractères avec majuscules, chiffres et symboles

3. Vérifiez les appareils connectés à votre compte

Facebook vous permet de voir tous les appareils actuellement connectés à votre compte. Si vous voyez un appareil ou une localisation que vous ne reconnaissez pas, votre compte a peut-être été compromis.

  1. Allez dans Paramètres → Sécurité et connexion
  2. Regardez la section "Où vous êtes connecté"
  3. Cliquez sur "Voir plus" pour voir tous les appareils
  4. Cliquez sur les 3 points à côté de chaque session suspecte → "Déconnexion"
  5. Si vous voyez des appareils inconnus, cliquez sur "Déconnexion de toutes les sessions" immédiatement
⚠️ Alerte :

Si vous trouvez des connexions depuis des pays étrangers (Maroc, Nigeria, Russie, Chine...) que vous n'avez pas faites, changez immédiatement votre mot de passe et activez le 2FA.

4. Activez les alertes de connexion

Facebook peut vous prévenir par email ou notification dès qu'une nouvelle connexion est détectée sur votre compte depuis un appareil inconnu.

  1. Allez dans Paramètres → Sécurité et connexion
  2. Cherchez "Recevoir des alertes concernant les connexions non reconnues"
  3. Activez les notifications par email et Facebook

5. Vérifiez les applications tierces connectées

De nombreuses applications demandent l'accès à votre Facebook ("Se connecter avec Facebook"). Certaines peuvent collecter vos données à votre insu. Faites le ménage régulièrement.

  1. Allez dans Paramètres → Applications et sites web
  2. Passez en revue toutes les applications listées
  3. Supprimez celles que vous n'utilisez plus ou que vous ne reconnaissez pas

6. Configurez les paramètres de confidentialité

Réduisez la visibilité de vos informations personnelles pour limiter les risques d'ingénierie sociale (technique où un hacker utilise vos infos publiques pour vous cibler).

  • Qui peut voir vos publications ? → Amis (pas "Public")
  • Qui peut vous trouver avec votre email/téléphone ? → Amis seulement
  • Votre liste d'amis → Visible uniquement par vous ou vos amis
  • Votre date de naissance → Masquez l'année

Pour accéder à ces paramètres : Paramètres → Confidentialité → Vérification de la confidentialité

7. Méfiez-vous des tentatives de phishing Facebook

Les hackers envoient de faux emails imitant Facebook : "Votre compte va être désactivé", "Une connexion suspecte a été détectée"... Ces emails vous dirigent vers de faux sites pour voler vos identifiants.

Les vrais emails de Facebook proviennent uniquement de @facebook.com ou @facebookmail.com. Ne cliquez jamais sur un lien dans un email suspect — allez directement sur facebook.com.

8. Protégez votre email de récupération

Si un hacker accède à votre boîte email, il peut réinitialiser votre mot de passe Facebook. Votre email est donc la clé de tout. Assurez-vous que votre email est également sécurisé avec un mot de passe fort et le 2FA.

9. Que faire si votre compte Facebook est déjà piraté ?

Si vous n'arrivez plus à vous connecter ou si des activités suspectes ont eu lieu :

  1. Allez sur facebook.com/hacked
  2. Suivez le processus de récupération de compte
  3. Prévenez vos amis que votre compte a été compromis pour qu'ils ignorent tout message suspect venant de vous
  4. Vérifiez si des messages ont été envoyés depuis votre compte à votre insu

Récapitulatif : checklist sécurité Facebook

  • Double authentification activée (application d'authentification)
  • Mot de passe fort et unique
  • Sessions actives vérifiées et nettoyées
  • Alertes de connexion activées
  • Applications tierces vérifiées et nettoyées
  • Paramètres de confidentialité configurés
  • Email de récupération sécurisé

En appliquant ces mesures, votre compte Facebook sera protégé contre la grande majorité des tentatives de piratage. Partagez cet article avec vos proches — beaucoup de personnes ont leur compte Facebook compromis faute de ces précautions basiques.

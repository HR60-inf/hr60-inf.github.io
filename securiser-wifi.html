Comment sécuriser son réseau Wi-Fi

Par Informa-Technique R  ·  11 mai 2026  ·  7 min de lecture

Un réseau Wi-Fi mal sécurisé est une porte ouverte pour les hackers de votre quartier. Ils peuvent voler votre connexion, intercepter vos communications, accéder à vos appareils connectés ou même utiliser votre connexion pour des activités illégales dont vous seriez responsable. Dans ce guide, apprenez à sécuriser votre Wi-Fi à la maison en quelques étapes simples.

1. Changez le mot de passe Wi-Fi par défaut

Le mot de passe Wi-Fi fourni par votre opérateur (Orange, SFR, Bouygues, Free) est souvent imprimé sur une étiquette sous votre box. N'importe qui entrant chez vous peut le lire et se connecter. Changez-le immédiatement.

Pour accéder à votre box :

  1. Ouvrez votre navigateur et tapez 192.168.1.1 (ou 192.168.0.1)
  2. Connectez-vous avec les identifiants admin (souvent "admin / admin" ou indiqués sous la box)
  3. Cherchez la section "Wi-Fi" ou "Sans fil"
  4. Changez le mot de passe par un mot de passe fort d'au moins 12 caractères
💡 Bon mot de passe Wi-Fi :

Utilisez une phrase comme Maison-Bleue-2026! — long, mémorisable et difficile à deviner. Évitez votre nom, adresse ou date de naissance.

2. Utilisez le chiffrement WPA3 ou WPA2

Le chiffrement détermine comment vos données sont protégées sur le réseau. WEP est obsolète et piratable en quelques minutes. WPA2 est acceptable. WPA3 est le plus récent et le plus sécurisé.

Dans les paramètres Wi-Fi de votre box :

  1. Cherchez "Type de sécurité" ou "Chiffrement"
  2. Sélectionnez WPA3 si disponible, sinon WPA2-AES
  3. Évitez absolument WEP et WPA (anciens et non sécurisés)

3. Changez le nom de votre réseau (SSID)

Le nom par défaut de votre Wi-Fi (ex: "Livebox-A3B2", "SFR-5678") révèle votre opérateur et le modèle de votre box. Un hacker peut utiliser cette information pour cibler des failles spécifiques à votre équipement.

  • Changez le SSID pour un nom neutre qui ne révèle pas votre opérateur, votre nom ou votre adresse
  • Exemples : "ResauMaison", "WiFi_Bleu", "NetworkHome"
  • N'utilisez pas : "WiFi de Mohamed Dupont" ou "Appartement 12"

4. Changez les identifiants admin de votre box

La plupart des box utilisent des identifiants admin par défaut ("admin/admin" ou "admin/1234"). Si quelqu'un se connecte à votre Wi-Fi, il peut accéder à l'interface d'administration de votre box avec ces identifiants.

  1. Dans l'interface admin de votre box, cherchez "Mot de passe administrateur"
  2. Changez-le pour un mot de passe fort différent de votre mot de passe Wi-Fi
  3. Notez-le dans un endroit sûr (gestionnaire de mots de passe)

5. Vérifiez qui est connecté à votre Wi-Fi

Votre box liste tous les appareils actuellement connectés. Si vous voyez un appareil inconnu, quelqu'un utilise votre connexion sans autorisation.

  1. Connectez-vous à l'interface admin de votre box (192.168.1.1)
  2. Cherchez "Appareils connectés" ou "Carte réseau"
  3. Identifiez chaque appareil (téléphones, ordinateurs, TV, consoles...)
  4. Si vous voyez un inconnu → changez immédiatement votre mot de passe Wi-Fi
⚠️ Signes que quelqu'un vole votre Wi-Fi :

Connexion lente sans raison, box qui clignote la nuit quand vous n'utilisez rien, facture internet anormalement élevée.

6. Créez un réseau Wi-Fi invité

Quand des amis ou visiteurs veulent se connecter à votre Wi-Fi, ne leur donnez pas votre mot de passe principal. Créez un réseau "invité" séparé qui n'a pas accès à vos appareils locaux (imprimante, disque dur, caméras).

  1. Dans l'interface admin, cherchez "Réseau invité" ou "Guest network"
  2. Activez-le avec un mot de passe différent
  3. Assurez-vous que l'option "Isolation des clients" est activée (les invités ne peuvent pas voir vos appareils)

7. Désactivez le WPS

Le WPS (Wi-Fi Protected Setup) est une fonctionnalité qui permet de connecter des appareils avec un simple bouton ou un code PIN à 8 chiffres. Ce code PIN est vulnérable aux attaques par force brute et peut être cracké en quelques heures.

  1. Dans l'interface admin, cherchez "WPS"
  2. Désactivez-le

8. Mettez à jour le firmware de votre box

Comme votre téléphone, votre box a un système d'exploitation (firmware) qui reçoit des mises à jour de sécurité. Assurez-vous qu'il est à jour.

La plupart des box (Livebox, Freebox, BBox, SFR) se mettent à jour automatiquement. Vérifiez dans l'interface admin s'il y a des mises à jour disponibles.

9. Positionnez votre box intelligemment

Placez votre box au centre de votre domicile plutôt que près d'une fenêtre ou d'un mur extérieur. Cela limite la portée du signal Wi-Fi à l'extérieur de chez vous, réduisant les risques que des personnes dans la rue ou dans les appartements voisins tentent de se connecter.

Récapitulatif : checklist sécurité Wi-Fi

  • Mot de passe Wi-Fi changé (fort et unique)
  • Chiffrement WPA3 ou WPA2-AES activé
  • Nom du réseau (SSID) personnalisé
  • Mot de passe admin de la box changé
  • Appareils connectés vérifiés régulièrement
  • Réseau invité créé pour les visiteurs
  • WPS désactivé
  • Firmware de la box à jour

En appliquant ces 8 mesures, votre réseau Wi-Fi sera beaucoup plus difficile à pirater. La sécurité de votre Wi-Fi protège non seulement votre connexion internet, mais aussi tous les appareils connectés chez vous : téléphones, ordinateurs, caméras, télévisions et objets connectés.
